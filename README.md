# 📦 Crear paquetes en R

Este repositorio reúne una guía paso a paso para crear, documentar y publicar paquetes en R utilizando el ecosistema de `usethis`, `devtools` y `roxygen2`.

El objetivo es contar con una referencia práctica que pueda reutilizarse en futuros proyectos.

## 🎯 Contenido

El flujo de trabajo está organizado en scripts independientes:

| Script | Descripción |
|---------|-------------|
| `configuracion_inicial.R` | Preparación del entorno de desarrollo. |
| `git_github.R` | Configuración de Git y autenticación con GitHub. |
| `creacion_paquete.R` | Creación de la estructura base del paquete. |
| `git_en_paquete.R` | Inicialización de Git y publicación en GitHub. |
| `desarrollo_funciones.R` | Creación y prueba de funciones. |
| `documentacion.R` | Documentación automática con roxygen2. |
| `dependencias.R` | Registro de paquetes utilizados. |
| `verificacion.R` | Verificación del paquete antes de compartirlo. |

## 📊 Flujo de trabajo 

1. Configurar el entorno de desarrollo.
2. Configurar Git y GitHub.
3. Crear el paquete.
4. Inicializar Git.
5. Publicar el repositorio.
6. Agregar funciones.
7. Documentar el paquete.
8. Registrar dependencias.
9. Ejecutar verificaciones.
